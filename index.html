T0F-00 (pronounced, tofu)

an open game project --- resources available on github:

Download as .zip Download as .tar.gz View on GitHub

Tofu, the robot, is looking for a safe home for his bomb - only to be hunted down by the intergalactic tofu blob everywhere he goes

download demo

controls:

standard FPS controls (WASD, Space bar, click-shoot)

don't let the bomb drop

don't let the enemy hit your bomb with tofu

don't hit your bomb with gunfire

don't fall off the platform

press ESC to quit

best played with a mouse

system requirements:

  • OS: Mac OSX (Windows, Linux coming soon)
  • HD usage (demo): 138.5 MB

Installation:

  • download the demo app
  • unzip it
  • put the file wherever you want (I recommend the Applications folder on Mac)
  • launch the app (double click)

about the project:

this game is created entirely with open source software and resources (except 3D modelling). due to the blender game engine license, this game is available for free! But it does take considerable effort and time to create, so please donate to see regular updates, and further game releases.

modelling and code: Blender 3D

textures, sounds, music: Open Game Art
